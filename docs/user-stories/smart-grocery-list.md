# User Story 007 - Liste de courses intelligente

Quand l'utilisateur sauvegarde des recettes dans Karnet, l'agent comprend les ingredients, les quantites, le nombre de portions, et les etapes importantes.

Plus tard, quand l'utilisateur choisit plusieurs recettes pour la semaine, Karnet peut generer une liste de courses consolidee.

L'agent regroupe les ingredients similaires, ajuste les quantites selon le nombre de personnes, et peut separer les produits par categorie: frais, epicerie, boissons, surgele, hygiene ou autre.

Karnet peut aussi detecter les ingredients probablement deja disponibles, comme sel, poivre, huile ou sucre, mais il doit laisser l'utilisateur les ajouter facilement si besoin.

## Display

Karnet affiche une liste de courses claire avec:

- les ingredients regroupes;
- les quantites totales;
- les recettes liees a chaque ingredient;
- un controle pour changer le nombre de personnes;
- des cases a cocher pendant les courses.

## Actions

Karnet peut preparer une liste partageable ou exportable, mais il demande confirmation avant de modifier une app externe, envoyer la liste, ou creer une commande.
