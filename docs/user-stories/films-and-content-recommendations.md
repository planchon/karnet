# User Story 012 - Films et contenus a regarder plus tard

L'utilisateur peut sauvegarder des films, series, videos, articles, podcasts ou recommandations trouvees dans une conversation, une app, un site, un Reel ou un message.

Quand le contenu est ajoute a Karnet, l'agent comprend le type de contenu, le theme, la duree approximative, la langue, le ton, et le contexte dans lequel l'utilisateur pourrait vouloir le consommer.

Plus tard, Karnet peut proposer un contenu adapte au moment: un film pour une soiree libre, une video courte pendant une pause, un podcast pour un trajet, ou un article pour un moment calme.

Karnet peut aussi faire remonter les contenus sauvegardes mais jamais consommes, pour eviter que la memoire devienne une pile morte.

## Display

Karnet affiche une carte avec:

- le titre;
- le type de contenu;
- la duree estimee;
- la source originale;
- une courte raison de la suggestion;
- une action pour ouvrir, marquer comme vu, garder pour plus tard ou oublier.

## Learning

Si l'utilisateur ignore souvent certains formats ou certains tons, Karnet doit adapter ses suggestions. Par exemple, proposer moins de podcasts si l'utilisateur ne les ecoute jamais, ou garder les films pour les soirs libres.
