# User Story 001 - Restaurants reels
When the user scrolls Instagram, he can find some wheels that it will want to share to Karnet. 
When the user shares a reel to Carnet, the agent understands what the reel is about and stores it in a special category related to the reel.
Nothing is overclocked by the backend; it's the agent doing its own work, managing its memory and tagging whatever it wants.

Then, for example, if the reel is a restaurant and the users come by the restaurant and it's lunch hours, the Karnet agent could send a notification to the users proposing that he try the restaurant.

When the user is looking for something to eat and it's near one of the restaurants, the agent should propose the restaurant to him.

## Display
For this user story, for example, Karnet should display a card of the restaurant with photos found on the internet, with descriptions, maybe a menu, the pricing, and how to get there.
