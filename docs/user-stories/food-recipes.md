# User Story 002 - Food recipes
When the user scrolls Instagram, for example, he could find some pretty good recipes that he would want to try later.
Sending the reel to Carnet will make Carnet understand the reel, understand the recipe, and save it in some kind of recipe database.

Then later, for example, when we are planning what to eat in the week, Karnet could propose some recipes that the user has saved but never really tried and try to come up with a meal plan for the week.

## Display
This display should be something simple
- A photo of the dish
- The link to the Instagram Reel
- The recipe
- All of the ingredients

Maybe with an input where I can choose the proportion depending on how many people will eat.
